# Sales Analytics System

A complete end-to-end **Sales Analytics application** built in Python that performs:

- Data ingestion with encoding handling  
- Data cleaning, validation, and filtering  
- Sales analysis and insights  
- External API enrichment (DummyJSON)  
- Report generation (text-based analytics report)

---

## Features

- Robust file reading with encoding fallback
- Data validation and optional user-driven filtering
- Revenue, region, product, customer, and trend analysis
- API-based product enrichment
- Detailed formatted report generation
- Graceful error handling and user-friendly console output
